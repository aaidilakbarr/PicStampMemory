# Framory

## Overview

Framory is a modern mobile-first memory journal application.

Instead of storing photos in a traditional gallery, users collect moments as beautiful postage stamps.

Each memory contains not only an image but also emotions, notes, dates, and places.

The goal is to make remembering moments feel nostalgic rather than simply browsing files.

---

## Vision

Create a gallery where every photo feels meaningful.

Users should feel like they are collecting memories instead of managing storage.
